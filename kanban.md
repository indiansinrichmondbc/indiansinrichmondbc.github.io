# Kanban Board

Manage your work using this markdown Kanban board. Move tasks between columns as they change status. You can add, edit, or complete items by updating this file, or ask Cline to do it for you.

---

## To Do

- [ ] Example: Add new business to directory
- [ ] Example: Update README with project description
- [ ] Example: Review privacy policy

## In Progress

- [ ] Example: Draft Holi event announcement

## Done

- [x] Set up Kanban board

---

**How to use:**
- Add new tasks under **To Do** (`- [ ] Task name`)
- Move a task to **In Progress** when you're working on it
- Move to **Done** and change `[ ]` to `[x]` when finished
- You can also ask Cline to add, move, or complete tasks for you—for example:  
  - "Add 'Prepare newsletter' to To Do"  
  - "Move 'Draft Holi event announcement' to Done"
